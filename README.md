# Mexico Real Estate Price Analysis

This project analyzes a dataset of 21,000 real estate listings in Mexico obtained from Properati.com.

The objective is to investigate whether property prices are more strongly influenced by property size or by geographic location.
Using data analysis and visualization techniques, the project explores relationships between price, 
surface area, and location to identify the factors that most significantly affect real estate prices.

Key questions:
- Does a larger property always mean a higher price?
- Do certain locations command higher prices regardless of size?
- Which variable explains more variation in price: size or location?

Tools used:
- Python
- Pandas
- Matplotlib
- plotly.express

The analysis is available in the Jupyter Notebook.
